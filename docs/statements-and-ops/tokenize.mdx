---
title: "Tokenize"
---

`TOKENIZE(X, **kwargs)`: This is an element-wise column operation. X should be a
column of texts. This operation tokenizes long texts into token windows of a
given length with a given stride. Alternatively, one can specify the exact
number of splits to tokenize the long texts into. This operation returns an
array-valued column that can be exploded with an unnesting operation.

### Args

<ParamField query="window" type="int" default="1">
  The length of the tokenization window.
</ParamField>

<ParamField query="stride" type="int" default="1">
  The stride of the tokenization.
</ParamField>

<ParamField query="splits" type="int" default="-1">
  An alternative way to specify the number of splits the tokenizer produces.
</ParamField>

### Examples

```sql
SELECT TOKENIZE(call_transcript) FROM calls_table;
SELECT UNNEST(TOKENIZE(call_transcript)) FROM calls_table;
```
