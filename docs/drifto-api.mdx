---
title: "Drifto API"
description: "Drifto's SQL interprets standard SQL 2003 statements and operations. Additionally, Drifto supports the following statements and operations."
---

General form of a SQL data selection query:

- `SELECT <SELECTOR>`
- `FROM <TABLE>`
- `JOIN <TABLE>`
- `ON <COLUMN>`
- `WHERE <CLAUSE>`
- `GROUP BY <COLUMN>`
- `HAVING <CLAUSE>`
- `ORDER BY <COLUMN>`
- `LIMIT <INT>;`
